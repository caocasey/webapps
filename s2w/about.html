Sense2Web: A Platform for Publishing Linked Data for the Internet of Things

  • Sense2Web supports the publication of flexible and interoperable sensor data descriptions.
  • The platform links sensor data/metadata to domain knowledge and represents the sensor devices using semantic descriptions.
  • The semantic descriptions are published as linked-data.
  • Publishing sensor data as linked data enables data to be related to domain knowledge and other relevant data and facilitates interconnection and integration of data from different sources.
  • Sense2Web publishes sensor device, real world entity and service descriptions as linked data and provides standard SPARQL query interfaces.
  • The platform provides a mash-up application to demonstrate browsing and accessing the linked sensor data and resources on a map.
  • The platform also provides RESTful interfaces for resource access, publication, and query support.
  • The platform currently adopts the IoT-A ontology models for describing IoT objects. The ontology models can be found at here and a journal paper explaining them can be found here
